# Rescussitou-iOS

<img src="images/icon.png" width=250px>

The iOS app for the Brazilian community of singers of the Neocatechumenal Way.

## Main features:
- Listing songs;
- Filtering songs;
- Searching songs;
- Viewing a song;
- Listening to the audio of the songs;
